# 🎓 Calculadora de Média Escolar

Um programa simples desenvolvido em **C** para calcular a média aritmética de duas notas.

## 🚀 Funcionalidades
- Recebe duas notas do utilizador.
- Calcula a média final.
- Exibe o resultado no terminal.

## 🛠️ Tecnologias
- Linguagem C
- Git & GitHub (Controlo de versão)

## 📖 Como executar
Para compilar o código, utilize um compilador de C (como o GCC):
```bash
gcc media-escolar.c -o media-escolar
./media-escolar
